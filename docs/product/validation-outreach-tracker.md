# Validation Outreach Tracker

Use this tracker to validate whether recruiters have strong enough pull for Talent Rediscovery before building beyond the lightweight demo or concierge workflow.

This is not a CRM. Keep one row per outreach target or conversation, update it manually, and only capture what helps decide whether to continue.

## Field Guide

| Field | Use |
| --- | --- |
| Segment | Freelance recruiter, small agency recruiter or founder, talent acquisition consultant, internal HR, or another explicit segment. |
| Source | Where this target came from, such as warm intro, LinkedIn, community, previous customer, or agency website. |
| Outreach status | Not contacted, contacted, replied, scheduled, interviewed, no response, declined, or follow-up later. |
| Current workflow | Where their Talent Pool lives today and how they currently rediscover Candidate Records for a new Search Request. |
| Pain intensity | Low, medium, high, or urgent, with a short note about the concrete recent pain. |
| Data access | Whether they would share an anonymized Talent Pool File, a sample export, a fake-but-realistic file, or only walk through their schema live. |
| Trust requirements | What they need to trust a Shortlist and its Matches, such as evidence from Candidate Records, gaps, risks, Search Criteria transparency, or manual review. |
| Pricing reaction | Reaction to a paid manual rediscovery report, Talent Pool audit, paid pilot, or beta subscription. |
| Next action | The single next manual step, such as send outreach, schedule interview, request anonymized file, run concierge test, ask pricing follow-up, or stop. |

## Counting Rules

Count an interview only after a real conversation about the recruiter's current workflow, pain intensity, data access, trust requirements, and pricing reaction.

Count a strong-pull signal only when the recruiter takes or explicitly requests a concrete next step toward testing, sharing data, introducing others, or paying. A polite positive comment is not enough.
